## How to run Locally to Contribute

1. Fork

2. `git clone <your-fork>`

3. Install dependencies

```bash
composer update
npm install
```

4. Run Website

```sh
cd pehapkari.cz
gulp
```

And open [http://localhost:8000](http://localhost:8000) in your browser.

That's all!
